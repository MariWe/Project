Teststrecke für das FusionSystems Parkhaus

Auf dieser Seite können Sie eine AR-Teststrecke ablaufen.
Diese Navigation ist eine AR-Anwendung, das heißt, dass Sie die Route und
die Richtung, in die Sie gehen müssen, in die reale Umgebung eingeblendet bekommen.

Dafür muss die Anwendung auf Ihren Standort und Ihre Kamera zugreifen.
Wenn sie ablehnen, kann die Anwendung nicht getestet werden.

Viel Spaß!
